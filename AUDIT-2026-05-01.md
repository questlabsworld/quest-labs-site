## ­čö┤ Critical ÔÇö Accessibility (Font Sizes)

**60+ font-size violations** across every file. WCAG 2.1 minimum is 12px for any visible text. Several go as low as 8px and 9px.

### Worst offenders

| File | Sizes found | Count |

| `about.astro` | 0.6rem, 0.65rem | 3 |

### What to do

Most violations fall into two categories:

1. **Decorative labels / eyebrows** (e.g. `font-size: 0.6rem`) ÔÇö bump to `0.75rem` (13.5px). These are the small uppercase Space Mono labels above headings.
2. **Stat labels and node text** (e.g. `8px`, `9px` in global.css around lines 913, 1006, 1067) ÔÇö these may be inside SVG-like diagram elements. If they're informational, they must be at least 12px. If purely decorative, mark with `aria-hidden="true"`.

The cleanest fix is a global search-and-replace:
- `font-size: 0.6rem` Ôćĺ `font-size: 0.75rem`
- `font-size: 0.55rem` Ôćĺ `font-size: 0.75rem`
- `font-size: 0.52rem` Ôćĺ `font-size: 0.75rem`
- `font-size: 8px` / `9px` / `10px` Ôćĺ `font-size: 12px` (or add `aria-hidden` if purely decorative)

## ­ččí Medium ÔÇö SEO

**Every page is missing `<meta name="description">`.** Google pulls this into search result snippets. Without it, it scrapes random text ÔÇö usually the first sentence of the page, which may not be compelling.

Add to `Layout.astro` head (accepting a `description` prop):

```astro
---
interface Props {
  title: string;
  description?: string;
}
const { title, description = 'Quest Labs ┬Ě Healing-Centered Futures ┬Ě Facilitation, research, and sandbox sessions for educators and organizations.' } = Astro.props;
---

<meta name="description" content={description} />
```

Then each page can pass its own:
```astro
<Layout title="Quest Labs: About" description="Mak─ôda Gershenson ÔÇö practitioner-researcher, founder of Quest Labs and Mosaics in the Making. Futures facilitation, decolonial foresight, healing-centered practice.">
```

**No Open Graph or Twitter Card tags.** When shared on social (LinkedIn, Instagram, iMessage), the link will show no image, no description, and a raw title. Add to Layout.astro head:

```html
<meta property="og:title" content={title} />
<meta property="og:description" content={description} />
<meta property="og:image" content="/images/og-preview.png" />
<meta property="og:type" content="website" />
<meta name="twitter:card" content="summary_large_image" />
```
